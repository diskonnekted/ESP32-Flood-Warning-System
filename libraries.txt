# Required Libraries for ESP32 Flood Warning System

## Arduino Libraries (Install via Library Manager)

1. **Blynk Library**
   - Name: Blynk
   - Version: 1.3.2 or later
   - Author: Volodymyr Shymanskyy
   - Installation: Library Manager → Search "Blynk" → Install

2. **NewPing Library**  
   - Name: NewPing
   - Version: 1.9.7 or later
   - Author: Tim Eckel
   - Installation: Library Manager → Search "NewPing" → Install

3. **WiFi Library**
   - Name: WiFi (built-in ESP32)
   - Version: Built-in with ESP32 core
   - Note: No separate installation needed

## Additional Libraries (Manual Installation if needed)

4. **ArduinoJson** (untuk future enhancements)
   - Name: ArduinoJson  
   - Version: 6.21.3 or later
   - Author: Benoit Blanchon
   - Usage: Data parsing dan formatting

5. **TimeLib** (untuk timestamp)
   - Name: Time
   - Version: 1.6.1 or later  
   - Author: Paul Stoffregen
   - Usage: Real-time clock functions

## ESP32 Board Package

**ESP32 Arduino Core**
- Board Manager URL: https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
- Version: 2.0.14 or later
- Installation: 
  1. File → Preferences → Additional Board Manager URLs
  2. Add the URL above
  3. Tools → Board → Board Manager → Search "ESP32" → Install

## Library Installation Steps

### Method 1: Arduino IDE Library Manager
```
1. Open Arduino IDE
2. Go to Sketch → Include Library → Manage Libraries
3. Search for library name
4. Click Install on the latest version
5. Wait for installation to complete
```

### Method 2: Manual Installation
```  
1. Download library ZIP from GitHub
2. Sketch → Include Library → Add .ZIP Library
3. Select downloaded ZIP file
4. Library will be installed automatically
```

### Method 3: GitHub Clone (Advanced)
```bash
cd ~/Arduino/libraries/
git clone https://github.com/blynkkk/blynk-library.git
git clone https://github.com/teckel12/NewPing.git
```

## Code Includes
Add these includes at the top of your Arduino sketch:

```cpp
#define BLYNK_TEMPLATE_ID "TMPL6xxxxxxxx"
#define BLYNK_TEMPLATE_NAME "Flood Warning System"  
#define BLYNK_AUTH_TOKEN "xxxxxxxxxxxxxxxxxxxxxxxxxxx"

#include <WiFi.h>
#include <BlynkSimpleEsp32.h>
#include <NewPing.h>

// Optional libraries for advanced features
#include <ArduinoJson.h>
#include <TimeLib.h>
```

## Troubleshooting Library Issues

### Common Errors and Solutions

**Error: "Blynk.h not found"**
```
Solution: Install Blynk library via Library Manager
Check: Sketch → Include Library → Contributed Libraries
```

**Error: "NewPing.h not found"**  
```
Solution: Install NewPing library
Alternative: Download from GitHub and add manually
```

**Error: "WiFi.h not found"**
```
Solution: Install ESP32 board package
Check: Tools → Board → ESP32 Arduino
```

**Error: Compilation failed**
```  
Check: Library version compatibility
Update: All libraries to latest versions
Restart: Arduino IDE after installation
```

### Library Path Issues
```
Windows: C:\Users\[Username]\Documents\Arduino\libraries\
Mac: ~/Documents/Arduino/libraries/
Linux: ~/Arduino/libraries/
```
